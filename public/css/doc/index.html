Metas:
Con la nueva página web de la division argentina esperamos creasr una experiencia unica para los miembros de la división. La web tiene un concepto facil de usar y que al mismo tiempo proporciona todos los datos necesariospara el desarrollo de las activiadesde la division.
Funcionalidades:
  • Accesso de usuarios a áreas restringidas.
  • Contenido dinamico manejado por el Staff a nivel departamental.
  • Secretaria Web con:
    • Entrenamientos
    • Examenes
    • Seguimiento de solicitudes
    • Seguimiento de avanze
    • AR Elite Pilots
Importancia:
Esta página nos pone en el mismo nivel que mas divisiones mas grandes de IVAO, teniendo un sistema unico en su tipo que permite a automacion de tareas en un 80% y permite una facil operacion del contenido web para divisiones grandes y pequeñas.
